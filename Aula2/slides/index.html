Linguagens de Programação

  • As linguagens de programação se dividem em 2 grupos:
    • Baixo Nível: aquelas que oferecem pouca ou nenhuma abstração ao conjunto de instruções da arquitetura.
      • São Rápidas
      • Não há necessidade de compilar, pois já está no formato que o processador "entende"
    • Alto Nível: Apresentam alta abstração do conjunto de instruções do computador. Neste tipo de linguagem não é necessário que se conheça a arquitetura do computador (32 / 64 bits; CISC, RISC, etc).
      • Um pouco mais lentas que a de baixo nível
      • Precisa de um compilador (que traduz o programa para uma linguagem de baixo nível)

Linguagens de Programação

  • As linguagens de programação também se dividem com relação à execução do código fonte:
    • Compilada: aquelas que executam o código diretamente no Sistema Operacional ou Processador.
      • São Rápidas
      • Ex.: C, C++, Fortran, Pascal, Obj-C
    • Interpretada: necessitam de outro programa (interpretador) capaz de traduzir e executar o código fonte e executar os comandos no S.O ou Processador.
      • Mais lentas
      • Ex.: Java, JavaScript, Python, C#, ASP

JavaScript

  • Para declarar uma variável em JavaScript, utiliza-se a palavra var 
    <script type="text/javascript">
    var x;
</script>
  • O JavaScript possui 2 tipos nativos de dados:
    • Número: pode ser real ou inteiro 
      <script type="text/javascript">
    var x = 1.25;
</script>
    • String: é uma cadeia de caracteres (Deve estar entre "") 
      <script type="text/javascript">
    var x = "Meu Texto";
</script>
