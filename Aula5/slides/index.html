Objetos

  • Objetos são utilizados para representar entidades de um problema
  • O conceito foi criado a partir do paradigma de programação conhecido por Orientação a Objetos
  • Um objeto tem comportamentos e propriedades
  • Os comportamentos são definidos através de métodos (funções internas)
  • Os estados são representados por atributos

Objetos

  • Existem 3 formas de criação de objetos:
    • Forma literal
    • Função construtora
    • A partir de Object

Objetos

  • Forma literal 
    
            var meuObjeto = {
              placa: "XYZ-1234",
              marca: "Ferrari",
              ligar: function(){
                console.log("Ligando carro de placa: " + this.placa);
              }
            };
  • Para utilizar o objeto: 
    
                 meuObjeto.ligar();

Objetos

  • Função construtora 
    
            function Carro(placa,marca){
              this.placa = placa;
              this.marca = marca;
              this.ligar = function(){
                console.log("Ligando carro de placa: " + this.placa);
              }
            }
  • Para utilizar o objeto precisamos primeiro criar: 
    
                 var carro = new Carro("XYZ-1234", "Ferrari");
                 carro.ligar();

Objetos

  • Através da classe Object 
    
              var carro = new Object();
              carro.placa = "xyz-1234";
              carro.marca = "Ferrari";
              carro.ligar = function(){
                console.log("Ligando carro de placa: " + carro.placa);
              }
  • Para utilizar o objeto: 
    
              carro.ligar();

Exemplo de Uso de Objetos


          function Pessoa(nome, telefone, idade){
             this.nome = nome;
             this.telefone = telefone;
             this.idade = idade;
             this.isMenor = function(){
                return this.idade < 18;
               }
          }

          var agenda = {
            lista: [],
            adiciona: function(p){
              this.lista.push(p);
            },
            removePorNome: function(nome){
              for(var i in this.lista){
                if(this.lista[i].nome == nome){
                  this.lista.splice(i,1);
                }

            },
            imprime: function(){
              console.table(this.lista);
            }
          }
  • Nesse exemplo, Pessoa é uma função construtora e agenda é um objeto criado de forma literal

O tipo Object

  • É a base de qualquer objeto JavaScript
  • Define métodos e propriedades padrão
  • Contém funções estáticas para auxiliar a manipulação de qualquer outro objeto

O tipo Object

Alguns métodos importantes

  • Object.call(this_obj, parametros, ...) : o mesmo que chamar a função Object tendo this_obj mapeado para this. mais informações
  • function.bind(thisArg[, arg1[, arg2[, ...]]]): O método bind() cria uma nova função que, quando chamada, tem sua palavra-chave this definida com o valor fornecido, com uma sequência determinada de argumentos precedendo quaisquer outros que sejam fornecidos quando a nova função é chamada.mais informações
  • fun.apply(thisArg, [argsArray]): O método apply() chama uma função com um dado valor this e arguments providos como uma array (ou um objeto parecido com um array).mais informções

O tipo Object

Alguns métodos importantes

  • Object.assign(base, o1,o2,...): concatena os métodos e atributos de o1,o2,..., em base e retorna. mais informações
  • Object.create(proto, properties): cria um objeto tendo como base de protótipo proto e propriedades properties mais informações
  • Object.defineProperty(obj, prop, descriptor): adiciona uma propriedade a obj de nome prop utilizando o descritor descriptor mais informações

Herança em Javascript

Para entender herança, precisamos entender primeiro como os objetos funcionam em JavaScript

  • Todo objeto em JavaScript é um Object
  • Todo objeto em JavaScript contém as propriedades e métodos de um prototype
  • A mudança de um prototype afeta todos os objetos relacionados a ele.

Prototypes

Vamos para um Exemplo:


          function Pessoa(nome, idade){
            this.nome = nome;
            this.idade = idade;
            return this;
          }
          var p1 = new Pessoa("P1", 18);

          //Vamos adicionar um atributo no prototype de Pessoa
          console.log(p1.nacionalidade); //Não imprime nada

          Pessoa.prototype.nacionalidade = "Brasileiro";

          var p2 = new Pessoa("P2", 18);

          // as duas pessoas são Brasileiras
          console.log(p1.nacionalidade); 
          console.log(p2.nacionalidade);

A função Object.call

Agora sim: juntando tudo para fazer herança

  • Quando queremos fazer herança, temos que fazer 2 coisas:
    • Dentro do construtor da classe filha, chamados o construtor da classe pai
    • Clonamos o protótipo da classe pai para a classe filha

function Funcionario(nome, salario){
    this.nome = nome;
    this.salario = salario;

    this.getSalario = function(){
        return this.salario;
    }

    return this;
}


function Vendedor(nome, salario){
    //Chamando construtor da class Pai
    Funcionario.call(this, nome, salario);
    this.totalVendas = 0;

    this.novaVenda = function(valor){
        this.totalVendas += valor;
    }
}
//Clonando prototype da class pai
Vendedor.prototype = Object.create(Funcionario.prototype);




function VendedorComissionado(nome, salarioFixo, comissao){
    //Chamando construtor da class Pai
    Vendedor.call(this, nome, salarioFixo);
    this.comissao = comissao;

    this.getSalario = function(){
        return this.salario + this.comissao*this.totalVendas;
    }
}
//Clonando prototype da class pai
VendedorComissionado.prototype = Object.create(Vendedor.prototype);


//Testando
var f1 = new Funcionario("Margarete", 3420.00);
var f2 = new Vendedor("Leandro", 3500.00);
var f3 = new VendedorComissionado("Kelly", 3500.00, 1.25);

console.log(f1.getSalario());
console.log(f2.getSalario());
console.log(f3.getSalario());


f2.novaVenda(1000);
f3.novaVenda(1000);


console.log(f2.getSalario());
console.log(f3.getSalario());
